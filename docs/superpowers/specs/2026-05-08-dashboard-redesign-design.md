# 장애 보고 대시보드 UI 리디자인 설계

**날짜**: 2026-05-08  
**범위**: 색상 테마 전환, 차트 4개 완성, 반응형 레이아웃

---

## 목표

- 색상: 주황(`#ff6b35`) + 짙은 남색(`#1a1a2e`) 다크 테마
- 차트: 일자별 Bar, 심각도별 Doughnut, 상태별 Doughnut, 팀별 수평 Bar
- 반응형: 데스크톱 4열/3열, 모바일 2열 그리드

---

## 색상 시스템

| 역할 | 값 |
|---|---|
| body 배경 | `#0f0f1a` |
| 카드/패널 | `#1a1a2e` |
| 카드 보더 | `#2d2d44` |
| 주요 포인트 | `#ff6b35` |
| 보조 포인트 | `#e65a1e` |
| 텍스트 기본 | `#e8e8f0` |
| 텍스트 보조 | `#8888aa` |
| 헤더 배경 | `#0a0a14` |

---

## 레이아웃 구조

```
[헤더] 로고 + 탭 네비게이션 (active: 주황 포인트)

[KPI 그리드] 4열(데스크톱) / 2×2(모바일)
  전체 | 접수 | 처리중 | 완료

[일자별 Bar 차트] 전체 폭

[서브 차트 그리드] 3열(데스크톱) / 2열(모바일)
  심각도별 Doughnut | 상태별 Doughnut | 팀별 수평 Bar

[최근 장애 목록 테이블]
```

---

## 차트 색상

| 차트 | 색상 |
|---|---|
| 일자별 Bar | 주황 `#ff6b35`, hover `#e65a1e` |
| 심각도 Doughnut | 상 `#e63946`, 중 `#ff6b35`, 하 `#4a9eff` |
| 상태 Doughnut | 접수 `#8888aa`, 처리중 `#ff6b35`, 완료 `#2a9d8f` |
| 팀별 수평 Bar | `#ff6b35` |

---

## 반응형 브레이크포인트

| 구간 | KPI 그리드 | 서브 차트 |
|---|---|---|
| ≥1024px | 4열 | 3열 |
| 768~1023px | 2×2 | 2열 |
| ≤767px | 2×2 | 2열 (일자별만 전체 폭) |

---

## 변경 파일

| 파일 | 내용 |
|---|---|
| `style.css` | 전면 재작성 — 다크 테마, CSS 변수, 반응형 미디어 쿼리 |
| `src/dashboard.js` | 차트 3개 추가 (`renderSeverityChart`, `renderStatusChart`, `renderDeptChart`), trendChart 색상 업데이트 |
| `index.html` | KPI 카드 마크업 개선 (icon + label + value 재배치) |

## 제약 조건

- Vanilla JS + Chart.js 4 유지 (외부 라이브러리 추가 없음)
- Supabase 연동 로직 변경 없음
- `.env` 파일 수정 금지
